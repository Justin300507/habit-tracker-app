# Habit Tracker App

> Generated by **ForgeAI** — AI-powered full-stack application generator.

## Features

- Sign‑up, login, password reset, and JWT‑based session handling
- Create, edit, delete, and list daily habits per user
- Mark habits as done for the current day and store a log entry
- Compute and display consecutive‑day streaks for each habit

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | FastAPI |
| Database | SQLite |
| Frontend | React + Vite |
| Auth | JWT Bearer tokens |

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+ (for frontend)

### Backend

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload
```

API available at: `http://localhost:8000`
Interactive docs: `http://localhost:8000/docs`

### Frontend

```bash
npm install
npm run dev
```

Frontend at: `http://localhost:5173`

### Docker

```bash
docker build -t habit_tracker_app .
docker run -p 8000:8000 habit_tracker_app
```
